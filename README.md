# FUTURE_ML_01 - Sales & Demand Forecasting

## Objective
Build a model to forecast future sales using historical business data.

## Dataset
Superstore Sales Dataset (2015-2018)

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Kaggle Notebook
[View Notebook](https://www.kaggle.com/code/moni35/notebook32ec1d23d5)

## Results
- Analyzed monthly sales trends from 2015-2018
- Built Linear Regression forecasting model
- Predicted next 6 months sales successfully
